### Custom Scripts for selecting a wallpaper with Rofi
#

* `theme-selector.sh` - not implemented, yet.

* `wallpaper-selector.sh` - Openes a rofi window to select a wallpaper to change to.

* `wallpaper-theme-selector.sh` - Launches `wallpaper-selector` and also changes the active color scheme.

* `wallpaper-thumbnail-cache.sh` - It creates thumbnails for each wallpaper image, can run with `-f` flag to recreate the thumbnails.

The cache shuold be used with `-f` each time you add a wallpaper.
